**JOB-008: Hoàn thiện Module Auth & User Profile**

**1. Context:**
Hệ thống đã có luồng Đăng nhập/Đăng ký. Hiện tại cần hoàn thiện các chức năng quản lý phiên và hồ sơ cá nhân của người dùng dựa trên schema bảng `user` và `refresh_token`.

**2. Input:**
- Luồng sẵn có: Đăng nhập/Đăng ký.
- Bảng liên quan: `user`, `refresh_token`.
- Service liên quan: `IAuthService`, `ICurrentUserService`, `ICloudinaryService`.

**3. Output:**
- **Logout (Revoke Token):** Tạo endpoint `POST /api/auth/logout`. Nhận `refreshToken` từ body hoặc cookie, tìm token trong DB và gán `revoked_at = DateTime.UtcNow`.
- **Current User Profile (Me):** Tạo endpoint `GET /api/auth/me` (yêu cầu `[Authorize]`). Dùng `ICurrentUserService` lấy ID, truy vấn bảng `user` (bao gồm cả `Roles`) và trả về `UserProfileDto`.
- **Update Profile (Upload Avatar):** Tạo endpoint `PUT /api/auth/me` (yêu cầu `[Authorize]`, dạng `multipart/form-data`). Tái sử dụng `ICloudinaryService` (đã viết ở module Film) để upload ảnh chân dung vào cột `avatar_url`. Cho phép cập nhật `display_name` và `bio`.
- **Service cập nhật đồng bộ:** Cập nhật `IAuthService` và `AuthService` tương ứng với các chức năng mới.
- **Chuẩn response bảo mật:** Đảm bảo tất cả endpoint tuân thủ chuẩn `ApiResponse<T>` của hệ thống.
- **Yêu cầu đầu ra bổ sung:** Code các DTOs mới, cập nhật Controllers/Services và báo cáo triển khai.

**4. Ràng buộc:**
- Tuân thủ kiến trúc Layered hiện tại.
- Không sửa file trong thư mục `Entities`.
