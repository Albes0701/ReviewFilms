**JOB-006: Triển khai Module Notification (Core API)**

**1. Context:**
Xây dựng hệ thống lưu trữ và truy xuất thông báo trong ứng dụng (Chưa làm Real-time SignalR ở giai đoạn này).

**2. Input:**
- Entity: `Notification`.

**3. Output:**
- **DTOs (`/DTOs/Notifications`):** Tạo `NotificationResponse`.
- **Services (`/Interfaces` & `/Services`):** - Tạo `INotificationService` với các hàm: `CreateNotificationAsync`, `GetUserNotificationsAsync` (phân trang), `MarkAsReadAsync`.
- **Controllers (`/Controllers`):** Tạo `NotificationsController` bảo vệ bằng `[Authorize]`. Cung cấp API cho phép frontend pull danh sách thông báo của user đang đăng nhập và API đánh dấu đã đọc.

**4. Ràng buộc:**
- Dữ liệu `DataJson` trong Notification (nếu có) phải được trả về đúng định dạng object trong JSON response, không trả về chuỗi escape string.
