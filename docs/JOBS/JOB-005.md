**JOB-005: Triển khai Module Review (Rating & Nested Comments)**

**1. Context:**
Xây dựng tính năng đánh giá phim (1-10 điểm) và hệ thống bình luận đa cấp (Self-reference).

**2. Input:**
- Các Entities: `MovieRating`, `Comment`, `CommentVote`.

**3. Output:**
- **DTOs (`/DTOs/Reviews`):** Tạo `RatingRequest`, `CommentRequest`, `CommentResponse` (có danh sách `ChildComments` bên trong).
- **Services (`/Interfaces` & `/Services`):** - Tạo `IReviewService`: Logic thêm/sửa/xóa điểm đánh giá. Logic tính toán lại `AvgRating` của `Movie`.
  - Logic tạo bình luận (xử lý đúng `ParentId` và `RootId` để ra cấu trúc cây).
- **Controllers (`/Controllers`):** Tạo `ReviewsController` để nhận các request đánh giá và bình luận. Yêu cầu có `[Authorize]` để lấy ID user từ JWT Token.

**4. Ràng buộc:**
- Thuật toán lấy danh sách Comment phải tối ưu. Tránh query n+1 khi lấy danh sách comment con.
- Controller phải nhận biết được User nào đang gọi API thông qua Claims từ JWT token.
