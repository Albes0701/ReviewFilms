**Vai trò của bạn:** Bạn là Senior .NET Developer. Hãy đọc kỹ file `AGENT.md` tôi đã cung cấp để hiểu rõ quy chuẩn kiến trúc của dự án.

**JOB-001: Khởi tạo Base Architecture & Database Connection**

**1. Context (Bối cảnh):**
Chúng ta đang xây dựng dự án ReviewFilms API bằng ASP.NET Core 10.0 LTS. Cần thiết lập nền móng chuẩn chỉnh trước khi ánh xạ database.

**2. Input (Đầu vào):**
- Cấu trúc thư mục định nghĩa trong `AGENT.md`.
- Database: PostgreSQL.

**3. Output (Yêu cầu đầu ra):**
Vui lòng cung cấp mã nguồn (và hướng dẫn tạo file/folder tương ứng) cho các hạng mục sau:
- Tạo cây thư mục chuẩn xác như `AGENT.md` đã quy định.
- Cấu hình file `appsettings.json` và `appsettings.Development.json` (thêm chuỗi kết nối DB PostgreSQL giả định, cấu hình Serilog hoặc logging mặc định).
- Tạo một `GlobalExceptionMiddleware.cs` trong thư mục `/Middlewares` để bắt lỗi toàn cục, định dạng mọi lỗi trả về dưới dạng JSON thống nhất (VD: `{ "success": false, "message": "...", "errors": [] }`).
- Tạo một `ApiResponse<T>.cs` trong thư mục `/DTOs/Common` làm wrapper cho mọi API response thành công (VD: `{ "success": true, "data": { ... }, "message": "..." }`).
- Viết file `Program.cs` sạch sẽ, gọi các phương thức extension để đăng ký Middleware, Controllers, và Swagger.

**4. Ràng buộc kỹ thuật:**
- KHÔNG viết logic nghiệp vụ (Service) hay Entity ở bước này. Chỉ setup nền móng.
- Chú ý namespace thống nhất là `ReviewFilms.Api.*`.
- Sử dụng Constructor Injection cho middleware.
